- Далее создадим скрипт CheckOffline. Он будет проверять, сколько игрока не было в сети, и выводить эту информацию на только что созданный текст.

```c#

using System;
using UnityEngine;
using TMPro;

public class CheckOffline : MonoBehaviour
{
    private TextMeshProUGUI statusText;
    public static CheckOffline Instance { get; private set; }
    private void InitSingleton()
    {
        Instance = this;
    }

    private void Awake()
    {
        InitSingleton();
        CheckOfflineStatus();
    }

    private void CheckOfflineStatus()
    {
        TimeSpan ts;
        if (PlayerPrefs.HasKey("LastSession"))
        {
            ts = DateTime.Now - DateTime.Parse(PlayerPrefs.GetString("LastSession"));

            GameObject status = GameObject.Find("Status");
            statusText = status.GetComponent<TextMeshProUGUI>();
            statusText.text = string.Format("Status: Online \n Вы отсутствовали: {0} день(дней/дня), {1} час(а/ов), {2} минут(у/ы), {3} секунд(у/ы)", ts.Days, ts.Hours, ts.Minutes, ts.Seconds);
        }

        PlayerPrefs.SetString("LastSession", DateTime.Now.ToString());
    }
}

```

- Сам скрипт положим на объект MainCamera, чтобы он работал и проверим его работоспособность. Он будет показывать количество дней, часов, минут и секунд, которое прошло с момента последнего визита игрока в игру.

## Задание 3
### Предложить наиболее подходящий на ваш взгляд способ монетизации игры D.Picker. Дать развернутый ответ с комментариями.

Ход работы:

- Есть много способов и вариантов добавления монетизации в игру. И сейчас я расскажу о тех методах, которые подойдут именно для этой игры.

### Реклама 
- В играх можно делать несколько точек входа рекламы, и в версии данной игры, которая сделана сейчас, я вижу несколько точек входа, и оодна из них - главное меню. В таком случае реклама будет появляться сразу в нескольких случаях: когда игрок только зайдет в игру и когда игрок проиграет и вернеться в главное меню. Еще одни точки входа - реализовать появление рекламы в тех случаях, когда игрок будет выходить из меню достижений, настроек, лидерборда или других меню, которые могут быть добавлены в будущем. Проблем с частым появлением рекламы не будет, так как яндекс разрешает показ только одной рекламы за 3 минуты.

### Реклама за вознаграждение
- Если простой рекламы не стоит добавлять очень много в игру, так как игроку может это надоесть, то в случае рекламы за вознаграждение можно, и даже нужно добавлять больше точек входа, потому что игроки любят, когда им предоставляют выбор, а так же предлагают игровые блага. В данной игре можно сделать следующие точки входа рекламы за вознаграждение: После смерти можно предложить воскреснуть за просмотр рекламы, а так же увеличить полученные очки в 2 раза. 
- В одном из видео мы делали кнопку, которая просто активирует рекламу за вознаграждение в главном меню. Можно развить идею, добавить в игру валюту и дать ограничение игроку: за сутки у игрока будет возможность 5 раз нажать на кнопку, и после нажатия он может получить или не получить награду. Будет что-то вроде колеса фортуны. 
- Дальше можно пофантазировать, что еще будет возможным в будущем. Например, если мы добавим валюту, то нужно добавить магазин, в котором могут продаваться задний фон, скины (на дракона, яйца, персонажа и щит), а так же некоторые способности для персонажа. Некоторые вещи будут продаватсься за игровую валюту, а какие-нибудь бонусные предметы можно получить только за просмотр рекламы.
- За выполнение достижений можно давать игровую валюту, а за просмотр рекламы игрок будет получать в два раза больше.

### Внутриигровые покупки
- У Яндекс.Игр есть своя валюта - Яны. Можно добавить в магазин покупку некотрых предметов за данную валюту. Например, можно конвертирвать Яны в игровую валюту, либо покупать за них способности для персонажа, которые будут сильно помогать в игре. Такой вариант монетизации тоже полезен, потому что у игроков появляется вариативность. Они могут выбрать, как им удобнее будет покупать игровые блага: за игровую валюту или за Яны. В данном случае игроки с Янами не будут получать никакого приемущества против игроков без них, так как у всех игроков одинаковые возможости. Поэтому данное решение является честным по отношению ко всем игрокам. Единственное преимущество, которое можно дать игрокам с Янами - эпические скины, которые можно приобрести только за Яны и никак не повлияют на геймплей.
